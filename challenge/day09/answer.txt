1. React.js는 어떤 기술이며 왜 활용되나요?
React.js는 meta가 대규모 웹 서비스의 UI를 더 편하게 개발하기 위해 만들어진 기술입니다.
React.js는 컴포넌트 기반으로 UI를 모듈화 할 수 있고, 선언형 프로그래밍 방식으로 화면의 업데이트 구성이 쉽습니다.
또한 가상 dom을 사용해 화면 업데이트도 빠르게 진행된다는 특징이 있습니다.

2. React App은 어떻게 생성할 수 있나요?
`create-react-app`이라는 명령어를 통해 생성할 수 있습니다.
하지만 빌드 과정을 설정하는 과정이 복잡하기 때문에 vite라는 빌드 관리 도구를 사용합니다.
vite를 이용하는 경우 `npm create vite@latest`로 생성할 수 있습니다.

3. React App은 어떻게 구동되나요?
vite를 사용한 경우, vite를 통해 생성된 리액트 앱이 웹 서버에 생성되어 있습니다.
이때, `npm reun dev`라는 리액트 웹을 가동시키는 명령어를 터미널에 입력해주면 리액트 웹 서버가 동작합니다.
기본적으로 리액트는 포트번호 3000이지만, vite로 생성한 경우 예외적으로 포트번호 5173으로 접속됩니다.
