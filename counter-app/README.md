## 구현 기능

**폴더구조**

```
├─ counter-app
│  ├─ README.md
│  ├─ eslint.config.js
│  ├─ index.html
│  ├─ package-lock.json
│  ├─ package.json
│  ├─ public
│  │  └─ vite.svg
│  ├─ src
│  │  ├─ App.css
│  │  ├─ App.jsx
│  │  ├─ assets
│  │  ├─ components
│  │  │  ├─ Controller.jsx
│  │  │  └─ Viewer.jsx
│  │  ├─ index.css
│  │  └─ main.jsx
│  └─ vite.config.js
```
